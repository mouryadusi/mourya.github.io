  • Easy to implement
  • No extra hardware
  • Works universally

√ó Cons

Monitor home Wi-Fi connection. Trigger reminder immediately when phone disconnects from network.

‚úì Pros

  • Instant detection
  • Very battery efficient

  • Fast response time

√ó Cons

  • False positives possible
